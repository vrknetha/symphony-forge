# @symphony/web

React + Vite frontend for the scaffolded project.

## Setup

```bash
pnpm install
```

## Development

```bash
pnpm dev        # dev server at localhost:5173
pnpm build      # production build
pnpm preview    # preview production build
```

## Testing

```bash
pnpm test       # unit + component tests (100% coverage required)
```

## Stack

React 19 · Vite · TanStack Query · Zustand · react-hook-form + Zod · Tailwind CSS · shadcn/ui
